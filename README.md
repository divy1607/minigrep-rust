# Rust Grep Mini

A lightweight version of the `grep` utility built in Rust. This tool searches for a specific word within a `poem.txt` file and saves the matching lines to `output.txt`.

## Getting Started

To get this project running locally, follow these steps:

1. **Clone the repository** to your local machine.
2. **Install Cargo** (the Rust package manager) if you haven't already.
3. **Run the program** using the following syntax:

```bash
cargo run -- {word_to_search} poem.txt
